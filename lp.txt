O que é lógica de programação?
	Uma sequência de instruções para atingir determinado objetivo

Hardware 
	Parte física do sistema

Software
	Parte lógica do sistema

Processamento de Dados
	Entrada - Processamento - Saída

Tipos de linguagem de programação
	Máquina -> bits [0] - [1] 
	Baixo Nível -> Assembly, compreensível pelos humanos
	Alto Nível -> Python, C, JS, Ruby....

Tipos de Dados
	Numéricos - Inteiro: 1,2,3,-1 | Real: 1.0, 2.00, 10.0
	Caracteres - letras, números e símbolos, regra: ('' ou "") <- obrigatoriamente 
	Lógicos - Verdadeiro ou Falso 

Variáveis
	Características:
		Identificação
		Tipo de dado
		Tamanho
		Valor (conteudo)
Nomes de variáveis
	Um ou mais caracteres
	Sempre letra primeiro
	N pode ter espaço em branco
	Zero símbolos
	Podem ser usados numeros
